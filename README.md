# Gashi Grand Complex

Kompleksi më modern i banimit në Prizren, Kosovë.

## Teknologjitë e përdorura

Ky projekt është ndërtuar me:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Framer Motion

## Si të ekzekutoni projektin

### Kërkesat

- Node.js & npm - [instaloni me nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

### Hapat

```sh
# Hapi 1: Instaloni varësitë
npm install

# Hapi 2: Nisni serverin e zhvillimit
npm run dev
```

## Komandat e disponueshme

- `npm run dev` - Nis serverin e zhvillimit me auto-reload
- `npm run build` - Ndërton projektin për prodhim
- `npm run preview` - Parapamje e build-it të prodhimit
- `npm run lint` - Kontrollon kodet për gabime

## Struktura e projektit

```
src/
├── components/     # Komponentët React
├── pages/         # Faqet e aplikacionit
├── hooks/         # Custom hooks
├── lib/           # Utilities dhe helpers
└── assets/        # Imazhet dhe resurset statike
```
